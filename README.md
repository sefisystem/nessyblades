# NESSY BLADES - Premium Barber Reservation SPA

A fully responsive, premium barber reservation single-page application (SPA) built with **React**, **Tailwind CSS**, and **Lucide React**.

## Features

✂️ **Premium Black & White Aesthetic**
- Clean, minimalist design with bold typography
- High-contrast color scheme for maximum readability
- Smooth animations and transitions

📱 **Fully Responsive**
- Mobile-first design
- Works perfectly on all screen sizes
- Touch-friendly interface

🎯 **Advanced Booking System**
- Name, phone, and email inputs
- Three service options with pricing
- Time slot picker (9 PM - 10 AM)
- Date selection
- Location input with dynamic map display

🗺️ **Interactive Map Component**
- Slides down dynamically when location is entered
- Realistic map mockup with grid effect
- Zoom controls
- Location marker

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Visit `http://localhost:3000` in your browser.

## Build

```bash
npm run build
```

## Deployment

This project is configured for Netlify deployment with a `netlify.toml` file.

Push to your repository and connect it to Netlify for automatic deployment.

## Tech Stack

- **React 18.2** - UI library
- **Tailwind CSS** - Utility-first CSS framework
- **Lucide React** - Icon library
- **Vite** - Build tool and dev server
